# Code of Conduct

## Purpose

Screenshot Maschine 3000 is intended to help people preserve evidence from messenger accounts they control. Discussions in this project should stay constructive, careful with private information, and respectful of people who may be using the app in stressful situations.

## Expected Behavior

- Be respectful and assume good intent.
- Keep feedback specific, actionable, and about the work.
- Respect privacy. Do not post private messages, screenshots, phone numbers, account IDs, auth data, or other personal information.
- Use the project only for lawful, authorized evidence preservation.

## Unacceptable Behavior

- Harassment, threats, insults, personal attacks, or discriminatory language.
- Publishing someone else's private information without explicit permission.
- Requests or contributions that enable stalking, hidden monitoring, credential theft, account compromise, or other abusive use.
- Sharing exploit details or security issues publicly before they can be handled safely.

## Reporting and Enforcement

If you see behavior that violates this code, contact the maintainer through the contact information on the maintainer's GitHub profile. If the issue can be handled publicly without exposing private information, you may also open a GitHub issue.

Maintainers may edit or remove comments, close issues or pull requests, reject contributions, or block users when behavior is harmful or out of scope for the project.

## Scope

This code applies to project spaces, including issues, pull requests, discussions, and any other official project communication.
